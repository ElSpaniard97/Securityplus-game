# Security+ Chapter 1 Learning Game (GitHub Pages)

This is a simple, single-page HTML learning game for Security+ Chapter 1 concepts:
- Flashcards
- Multiple choice quiz
- Matching game
- Score + streak tracking (saved in browser localStorage)

## Run locally
1. Download or clone this repo
2. Open `index.html` in your browser

## Host on GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings â†’ Pages**
3. Select:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
4. Your site will be available at:
   `https://<username>.github.io/<repo-name>/`

## Notes
- Stats are stored in your browser using localStorage.
- If you clear site data, stats reset.
